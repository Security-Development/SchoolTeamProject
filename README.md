# SchoolTeamProject

사용된 내용 : 최소 제곱법, 선형회귀

```
조원 : 이승용, 최인근, 황성윤
개발 : 이승용
제안서 : 최인근, 황성윤
발표자 : 미정
개발 상태 : 진행중
```
