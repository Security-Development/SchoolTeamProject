# SchoolTeamProject
Python 사이킷 런 모듈로 회귀 분석하여 다음날 코로나 감염자 선형회귀 분석

사용된 내용 : 최소 제곱법, 선형회귀

```
조원 : 이승용, 최인근, 황성윤
개발 : 이승용
제안서 : 최인근, 황성윤
발표자 : 미정
개발 상태 : 완료
```
